ls - 디렉토리 파일 목록을 보여준다. 사용 방법 ls
mv - 파일을 다른 디렉토리로 이동시키거나 이름을 변경한다. ls 파일 파일
cp - 파일의 복사본을 만든다. cp 파일 파일
chmod - 파일의 권한을 변경한다. chmod 권한설정(8진수) 파일
cd - 디렉토리를 변경한다.  cd 디렉토리
vi - 입력창을 띠워준다. vi 파일이름
cat - 현재 파일의 내용을 보여준다. cat 파일
rm - 파일을 제거한다. rm 옵션 파일
kill - 프로세스를 중지 시킨다. kill pid
ps - 프로세스 목록을 보여준다. ps 옵션
