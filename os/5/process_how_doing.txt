실행 파일이 실행되는 과정

마우스를 움직이면 인터럽트가 발생한다. 해당 인터럽트가 발생하면 마우스 드라이버는 마우스 위치 포인터와 상태(클릭)을 반환한다.
해당 파일 아이콘을 더블클릭 또는 콘솔창에 실행파일명을 적어 실행 동작 명령을 내린다.
os를 vfork를 하고 exec을 실행한다.
vfork시 task_struct를 구성할 최소한의 정보(inode정보, file operation등)를 복사한다. 이 때 해당 task가 runqueue에 올라간다.
exec함수에 실행시킬 파일의 경로와 파일 명이 입력된다.
exec 동작은 다음과 같다.
pathbuf 경로 문자열을 저장하는 변수
linux_binprm은 bin파일을 읽어오기 위한 변수
file은 현재 파일

inode를 참고하여 하드디스크 주소에서 실행파일 메모리에 올리기위한 사전 준비를 시작한다.
먼저 파일의 해더(128 Byte)를 읽어서 해당 파일이 어떤 파일인지 파악한다.
해당 파일이 elf임을 알았다면 fileoperation 구조체의 read 함수 포인터를 bm_entry_read함수와 연결시킨다.
bm_entry_read함수가 동작하여 실행파일 읽어온다.
